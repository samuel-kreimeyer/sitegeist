Installation Instructions

1

Check Your Chrome Version

Make sure you have Google Chrome version 141 or higher installed.

2

Download Sitegeist

Click the download button below to get sitegeist-latest.zip

Download Sitegeist
3

Unzip to a Known Location

Extract the zip file to an easy-to-find location. For example:

  • Windows: C:\Users\YourName\Desktop\sitegeist
  • macOS: ~/Desktop/sitegeist
  • Linux: ~/Desktop/sitegeist

After unzipping, you should have a folder named sitegeist containing the extension files.

4

Open Chrome Extensions Page

Open Chrome and type chrome://extensions in the address bar, then press Enter.

5

Enable Developer Mode

Toggle the "Developer mode" switch in the top right corner of the extensions page.

6

Load Unpacked Extension

Click the "Load unpacked" button that appears, then select your Desktop/sitegeist folder.

7

Configure Extension Settings

Locate the Sitegeist extension in "All extensions" and click "Details". Make sure the following options are enabled:

  • Allow user scripts
  • Pin to toolbar
  • Allow access to file URLs
8

Open Sitegeist

Use the keyboard shortcut to open/close Sitegeist:

  • macOS: CMD + SHIFT + S
  • Windows/Linux: CTRL + SHIFT + S
9

Enter API Token

On your first message submission, you'll be prompted to enter an API token. Contact Mario to get your token.

Privacy & Security

Your Data Stays Private

Settings & API keys: Stored only on your computer. Never sent anywhere.

Chat history: Stored only on your computer in IndexedDB.

When you send messages: Your message text and any attachments you include are sent to your chosen AI provider (default: Anthropic) to generate responses. These providers are configured to not retain your data or use it for training.

CORS proxy: ON BY DEFAULT. Needed for Anthropic OAuth tokens and Z-AI API due to CORS restrictions. Requests to the AI go through https://proxy.mariozechner.at/proxy which does not retain or log data. You can run your own CORS proxy or use a service like corsproxy.io

Why Does Sitegeist Need So Many Permissions?

Sitegeist needs special browser permissions to let the AI control your browser. Here's what each permission does in plain English:

  • Storage: Saves your sessions, settings, and API keys on your computer
  • Unlimited storage: Prevents the browser from deleting your saved sessions and settings
  • Active tab: Lets the AI see which website you're currently on
  • Side panel: Creates the Sitegeist chat interface
  • User scripts: Allows the AI to run code it generates on the current webpage (this is how it interacts with websites)
  • Web navigation: Lets the AI navigate to new URLs or go back/forward in your browser history
  • Debugger: For stubborn websites that block automated inputs, this creates real keyboard/mouse events. A banner appears at the top of your browser when this is active

Important: All these permissions are necessary for the AI to automate tasks on websites. Without them, Sitegeist can't do its job. Your data always stays on your computer unless you explicitly send it in a chat message.
